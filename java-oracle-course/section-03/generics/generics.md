- Use generic methods
- Use wildcards


## Enumerations (Enumerações)

- Enumerações (ou enums) são uma especificação para uma classe em que todas as instâncias da classe são criadas dentro da classe.
    − Enums são um tipo de dados que contém um conjunto fixo de constantes
    − Enums são bons para usar quando você já conhece todas as possibilidades dos valores ou instâncias da classe
    − Se você usar enumerações em vez de strings ou inteiros, você aumenta as verificações em tempo de compilação
- Pode ser declarado sozinho ou dentro outra classe
